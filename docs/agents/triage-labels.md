# Triage Labels

| Canonical 角色 | GitHub 标签 | 适用范围 |
|---|---|---|
| Product | `triage:product` | 用户问题、范围与验收 |
| Design | `triage:design` | UI、交互和视觉资产 |
| Engineering | `triage:engineering` | 实现、架构、依赖和迁移 |
| Quality | `triage:quality` | Bug、测试、性能与安全 |
| Docs | `triage:docs` | README、知识库与开发者体验 |

优先级另用 `priority:p0..p3`；不要把角色和紧急程度混成一个标签。
